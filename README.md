# ZEEG — Zap's Enhanced Enchantment Glints

Customize your enchantment glint color in Minecraft 1.21.11 without resource packs.

## Features

- Change enchantment glint color to any RGB value
- Hex color input (`#RRGGBB`)
- 6 presets: Cyan, Red, Green, Blue, White, Original
- Config screen accessible via Mod Menu
- Works with the glint strength slider in video settings

## Usage

1. Install the mod (Fabric, requires Fabric API)
2. Open Mod Menu → ZEEG → Configure
3. Pick a color and click Save & Reload
4. Glint color updates immediately

## Development

```
./gradlew build
```

Output is in `build/libs/zeeg-<version>.jar`.
